Git/Github

Git est un logiciel de gestion de fichiers décentralisé, crée par Linus Torvalds en 2005.

Sa particularité réside dans son organisation en arborescence, le dépôt original servant de tronc principal et les branches crées permettant de travailler indépendamment sur plusieurs versions du projet. Git permet de fusionner aisément plusieurs branches pour intégrer les avancées.

Il est le système de versioning le plus utilisé par les développeurs actuellement, réputé pour faciliter le travail en équipe et le développement de projets. Il est pensé pour être utilisé essentiellement en local.

Commandes de base

Installation de Git sous Linux

Télécharger la dernière version de Git pour le système d'exploitation de l'utilisateur sur : http://git-scm.com/downloads.

Suivre les instructions.

Entrer successivement les commandes ci-dessous dans le terminal pour définir le nom et l'e-mail qui seront utilisés sur Github.

git config [--global user.name] ["Votre nom ou pseudo"]

git config [--global user.email] ["Votre@email.com"]

Entrer la command "git" dans le terminal pour vérifier si l'installation s'est correctement effectuée. Une page d'instructions sur l'utilisation de Git devrait s'afficher dans le terminal.

Les commandes de création d'un dépôt

git init : Créer un dépôt Git local

git clone [adresseACloner] : Cloner un dépôt préexistant sur Github dans le répertoire de travail.

Commandes d'interaction avec des dépôts distants

git pull : Mettre à jour le dépôt local par rapport au dépôt distant.

git push : Envoyer les données du dépôt local dans le dépôt distant.

Les commandes de gestion des branches

git branch : Créer une nouvelle branche contenant une copie de la branche master.

git merge : Fusionner deux branches de développement.

Les commandes de gestions des fichiers

git add [nomDuFichier] : Ajouter le fichier en paramètre dans le repertoire de travail.

git commit -m ["message"] : Validation des modifications des fichiers dans le répertoire de travail.

git log : Afficher la liste des commits effectués.

git status : Afficher l'état courant du répertoire de travail.
